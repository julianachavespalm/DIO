O que é HTML?

HTML é uma linguagem de marcação que define a estrutura e o conteúdo de uma página web. Usando tags (etiquetas) e atributos, é possível organizar informações e elementos visuais para que sejam interpretados pelos navegadores.

O que são tags?

As tags são a base da estruturação do conteúdo HTML, permitindo a organização lógica e a formatação do texto, imagens, links e outros elementos em uma página da web. Elas são interpretadas pelo navegador para renderizar o conteúdo da página conforme indicado pelas instruções contidas nelas.

Elas são compostas pelo nome do elemento entre os símbolos < e >. Por exemplo, para criar um parágrafo, usamos a tag <p> para iniciar o parágrafo e </p> para fechá-lo. Dessa forma, o navegador identifica onde o parágrafo começa e termina, formatando corretamente o texto.

Algumas tags em HTML não requerem um fechamento explícito, como é o caso da tag <img>. Ela é usada para exibir imagens em uma página da web.

            
<img src="caminho/da/imagem.jpg" alt="Descrição da imagem" />

Note que a barra invertida ao final da tag indica que ela está completa em si mesma, não necessitando de um fechamento separado.

Estrutura básica de um site

Para estruturar um site, serão necessários basicamente quatro elementos principais do HTML: <DOCTYPE>, <html>, <head>, e <body>.

DOCTYPE - Document Type Declaration

O `DOCTYPE` é uma declaração que informa ao navegador sobre a versão do HTML usada no documento. Ele é essencial para garantir que o navegador interprete o código corretamente.

            <!DOCTYPE html>

HTML - HyperText Markup Language

O HTML é a base de qualquer página da web. Ele consiste em diferentes elementos, e três deles são fundamentais: <html>, <head> e <body>.

<html>: Define o Documento HTML

A tag <html> é o elemento raiz de um documento HTML. Ela engloba todo o conteúdo da página e define o início e o fim do documento.

            <!DOCTYPE html>
            <html lang="pt-BR">
                <head>
                    <!-- Metadados, como charset, título e links para estilos e scripts -->
                </head>
                <body>
                    <!-- Conteúdo visível da página, como texto, imagens, links -->
                </body>
            </html>

<head>: Metadados e Informações do Documento

A tag <head> contém informações sobre o documento, como o título da página, metadados, scripts, estilos e outras configurações.

        <head>
            <meta charset="UTF-8">
            <title>Título da Página</title>
            <link rel="stylesheet" href="estilos.css">
            <script src="script.js"></script>
        </head>

<body>: Conteúdo Visível da Página

A tag <body> contém o conteúdo visível da página, como texto, imagens, links, vídeos e outros elementos que os usuários visualizam.

        <body>
            <h1>Título Principal</h1>
            <p>Este é um parágrafo de exemplo.</p>
            <img src="imagem.jpg" alt="Descrição da Imagem">
            <a href="pagina.html">Link para Outra Página</a>
        </body>

Títulos

Para criar títulos em HTML, usamos as tags <h1> até <h6>. Elas representam diferentes níveis de importância, sendo o <h1> o mais importante e o <h6> o menos.

Além disso, para criar parágrafos, como citado anteriormente, basta utilizar a tag <p>.

Exemplo:

        <h1>Título</h1>
        <p>Este é um parágrafo de exemplo.</p>

Listas Ordenadas e Não Ordenadas

As listas em HTML podem ser ordenadas ou não ordenadas. As listas ordenadas são numeradas, enquanto as não ordenadas são marcadas com bullets.

Listas Ordenadas (ol)

        <ol>
            <li>Item 1</li>
            <li>Item 2</li>
            <li>Item 3</li>
        </ol>

Visualmente, o navegador interpretará como:

  1. Item 1
  2. Item 2
  3. Item 3

Listas Não Ordenadas (ul)

        <ul>
            <li>Item A</li>
            <li>Item B</li>
            <li>Item C</li>
        </ul>

Visualmente o navegador interpretará como:

  • Item 1
  • Item 2
  • Item 3

Links em HTML

Os links em HTML são elementos que permitem aos usuários navegar entre diferentes recursos da web, sejam outras páginas da internet, arquivos, seções na mesma página ou até mesmo endereços de e-mail. Os links são criados com a tag <a> em HTML.

Estrutura Básica de um Link:

Um link básico possui a seguinte estrutura:

        <a href="URL_do_destino">Texto do Link</a>

Atributo href:

O atributo mais importante de um link é o href, que determina o destino para onde o link aponta. Pode ser uma URL, um endereço de e-mail, um ID de elemento na mesma página, entre outros.

Exemplo de Link:

Um exemplo simples de link:

        <a href="https://www.exemplo.com">Visite Exemplo.com</a>

Link Interno na Mesma Página:

Para criar um link interno, utiliza-se o ID do elemento como destino:

        <a href="#section2">Ir para a Seção 2</a>

Abrir Link em Nova Aba:

Utilize o atributo target="_blank" para abrir o link em uma nova aba:

        <a href="https://www.exemplo.com" target="_blank">Visite Exemplo.com em uma nova aba</a>

Conclusão

O HTML é uma linguagem fundamental para a criação e estruturação de páginas da internet. Através de tags, como <p>, <h1>, <img> e outras, podemos organizar e formatar o conteúdo de maneira lógica e visualmente apelativa.

As tags são elementos entre colchetes angulares < e > que indicam ao navegador como exibir e estruturar o conteúdo da página. Com elas, é possível criar desde cabeçalhos e parágrafos até listas ordenadas e não ordenadas.

Além disso, os links em HTML, formados pela tag <a>, possibilitam a conexão entre diferentes recursos da web, permitindo a navegação entre páginas, seções na mesma página, endereços de e-mail e outros destinos, definidos pelo atributo href.

A compreensão básica desses conceitos fundamentais no HTML oferece um ponto de partida sólido para a construção de páginas web, permitindo estruturar conteúdos e criar links para uma experiência mais dinâmica e informativa na internet.
